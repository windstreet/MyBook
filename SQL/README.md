# SQL教程

# 一、什么是SQL？

- 简单地说，SQL就是访问和处理关系数据库的计算机标准语言。

- 无论用什么编程语言（Java、Python……）编写程序，只要涉及到操作关系数据库，都必须通过SQL来完成。


# 二、NoSQL

- 非SQL的数据库，包括MongoDB、Cassandra、Dynamo等等，它们都不是关系数据库。

- NoSQL数据库是作为SQL数据库的补充。
